# Book_Digital_Image_Processing
This repository contains the Python labs that accompany the book **Image Processing**. The goal is to guide students step-by-step from basic image representation to core image-processing operations.
## Quick start (recommended): Google Colab
No installation is required.

- Chapter 1 Lab (Basics): **Open in Colab**

> Tip: In Colab, click **File → Save a copy in Drive** to keep your work.

## Local setup (optional): VS Code / Jupyter
